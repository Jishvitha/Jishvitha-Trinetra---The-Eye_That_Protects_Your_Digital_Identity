# Trinetra – The Eye That Protects Your Digital Identity

Trinetra is a web-based application developed to help users protect their digital identity in the online world. It allows users to securely upload images or videos, generate a digital fingerprint, and track possible misuse of their content. The platform also provides guidance and support for people facing digital harassment or unauthorized use of personal media.

This project was created as part of an academic milestone to demonstrate a simple and practical solution for digital safety using web technologies.

## Features
- User registration and login  
- Secure media upload  
- Digital fingerprint generation (prototype)  
- Dashboard to view uploaded content  
- Support and safety guidance  

## Technology Used

- CSS  
- JavaScript  
- Browser Local Storage  
- GitHub
